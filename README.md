# TripSplit Lite

A simple web tool to split group expenses and generate settle-up suggestions.

## Demo
- Live: https://heulwenD.github.io/tripsplit-lite/

## Features
- Add members, add expenses (payer + split-with)
- Auto summary: Paid / Owed / Net
- Settle suggestions (who pays whom)
- LocalStorage persistence
- Export JSON / Import JSON
- Export CSV (expenses)

## How to run locally
Open `index.html` in a browser.

## Notes
This is a lightweight, frontend-only project (HTML/CSS/JS).
